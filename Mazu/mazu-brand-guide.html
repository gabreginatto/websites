Usage

CSS Variables

Include mazu-brand.css in your project to access all brand variables: 

/* Link the stylesheet */
<link rel="stylesheet" href="mazu-brand.css">

/* Use variables in your CSS */
.my-button {
    background: var(--mazu-gradient-primary);
    color: var(--mazu-white);
    border-radius: var(--mazu-radius-lg);
    padding: var(--mazu-space-sm) var(--mazu-space-lg);
}

JavaScript / React

Import brand tokens from mazu-brand.js: 

// Import brand tokens
import { colors, gradients, assets } from './mazu-brand.js'

// Use in your components
const Header = () => (
    <header style={{ background: gradients.primary }}>
        <img src={assets.logos.gradient.horizontal} alt="Mazu" />
    </header>
)
