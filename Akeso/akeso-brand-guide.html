Usage

CSS Variables

Include akeso-brand.css in your project to access all brand variables: 

/* Link the stylesheet */
<link rel="stylesheet" href="akeso-brand.css">

/* Use variables in your CSS */
.my-button {
    background-color: var(--akeso-primary);
    color: var(--akeso-white);
    border-radius: var(--akeso-radius-md);
    padding: var(--akeso-space-sm) var(--akeso-space-md);
}

JavaScript / React

Import brand tokens from akeso-brand.js: 

// Import brand tokens
import { colors, fonts, assets } from './akeso-brand.js'

// Use in your components
const Header = () => (
    <header style={{ backgroundColor: colors.primary }}>
        <img src={assets.logo} alt="Akeso" />
    </header>
)
